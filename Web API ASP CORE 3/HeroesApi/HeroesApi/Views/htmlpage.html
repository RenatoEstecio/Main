Exemplo API - ASP Net Core 3.0

Trata-se de uma API que usa REGEX para recuperar informações de uma pagina,
no caso deste exemplo: nerdologiaheroespowers.fandom.com/pt-br/wiki
é possível retornar atráves de uma API infinidades de tipos de arquivos, de mais comuns como xml e json,
ou arquivos como html, por exemplo este que você esta visualizando.


Exemplos:

Retornando Json: localhost:Porta/Api/Hq/NomeDoPersonagemMarvel

Retornando PDF: localhost:Porta/Api/Hq/pdf/NomeDoPersonagemMarvel
